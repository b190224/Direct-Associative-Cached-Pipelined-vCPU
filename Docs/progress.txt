	The data set is valid. Initially, all cache entries are invalid, so, a cache miss would occur and main memory needs to be searched for. However, 
	as soon as the instruction is fetched, three things are altered. One, the valid bit, which is set to true. Two, the tag, which is identified in the
	two least signifcant bits. And lastly, the instruction which in our case becomes four. This process repeats four times as four memory locations are
	mapped two one cache locations. The fifth entry could be thought of as a reset, all its fields are zero so the steps mentioned before are repeated.
